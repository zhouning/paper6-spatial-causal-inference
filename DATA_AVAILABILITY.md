# Data Availability and FAIR Notes

This repository is a reviewer reproduction package, not a public archival data
release. It contains code, manuscript source, synthetic generators, generated
result tables, and public/example data that can be redistributed under their
source terms. Restricted local inputs are documented but are not tracked in the
public GitHub tree.

## Chongqing Data Restriction

The Chongqing raw geospatial inputs and the 5,000-row building-level analysis
sample include precise building geometries or coordinates, floor attributes, and
derived environmental variables. They should not be redistributed on GitHub
unless the original data-use permissions and any applicable local security or
sensitivity review explicitly allow public release.

The current GitHub tree therefore removes those files from version control and
adds ignore rules for:

- `data/raw/`
- `paper/ijgis_submission_20260605/07_results/chongqing_uhi_analysis_sample.csv`

The local Windows workspace may still contain these files as untracked inputs so
the author can rerun the Chongqing case under an approved local workflow.

`CountyData` is third-party training/demo data, not author-generated data. The
metadata in `data/CountyData.shp.xml` identifies these sources and use terms:

## Metadata Checklist

- Dataset provenance is described here and in `MANIFEST.md`.
- Restricted Chongqing inputs are documented with a reason for non-public access.
- The county example carries explicit third-party source and use-restriction
  notes.
- Deterministic synthetic experiments use explicit random seeds in code.
- Generated figures, CSVs, JSON outputs, and Markdown reports are stored under
  stable paths.
- Checksums can be regenerated from a clean checkout after excluding restricted
  local data:

```powershell
Get-ChildItem -Recurse -File | Where-Object { $_.FullName -notmatch '\\.git\\|\\data\\raw\\|chongqing_uhi_analysis_sample\.csv$' } | Get-FileHash -Algorithm SHA256
```

## Recommended Final Submission Statement

For IJGIS submission, adapt this wording after confirming the final repository
URL and any reviewer-only access route:

> The code, manuscript source, synthetic generators, generated result tables,
> figures, and diagnostic reports supporting this study are available in the
> GitHub reproduction repository. Synthetic validation data are generated by the
> scripts in the repository. The Chongqing raw geospatial inputs and
> building-level UHI analysis sample are not publicly redistributed because they
> include precise building geometries/coordinates and are subject to source
> permission and sensitivity review; access can be evaluated through an approved
> institutional route for qualified reviewers or researchers. MODIS LST and
> other remote-sensing products can be re-extracted through Google Earth Engine
> by users with appropriate access. The county social-capital example uses
> third-party Esri training/demo data with attributes derived from CDC WONDER and
> County Health Rankings/Living Atlas sources; it is provided only under the
> source use restrictions documented in the dataset metadata.
