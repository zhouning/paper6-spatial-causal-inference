# Data Availability and FAIR Notes

This repository is a reviewer reproduction package rather than a public archival data release. It includes the data and generated artifacts needed to inspect and rerun the current Paper6 experiments.

## Access and Redistribution Flags

- The raw Chongqing geospatial files are included to avoid omitting Paper6 reproduction inputs. Before converting this repository to a public repository, confirm redistribution permissions for these files.
- Google Earth Engine-derived MODIS and AlphaEarth data require the reviewer to have GEE access for live re-extraction. The repository includes scripts and generated outputs, but not a full global AlphaEarth cache.
- No API keys, `.env` files, cookies, or internal GitLab remote credentials should be committed.

## Metadata Checklist

- File provenance is described in this document and in `MANIFEST.md`.
- Deterministic synthetic experiments use explicit random seeds in code.
- Raw geospatial inputs are preserved with shapefile sidecars and raster metadata files.
- Generated figures and JSON outputs are stored under stable paths.
- Checksums can be regenerated with:

```powershell
Get-ChildItem -Recurse -File | Get-FileHash -Algorithm SHA256
```

## Recommended Final Submission Statement

For IJGIS submission, adapt this wording after confirming raw-data permissions:

> The code, manuscript source, generated figures, selected reproduction data, and model artifacts supporting this study are available in a private reviewer repository. The synthetic validation data are generated by the scripts in the repository. The Chongqing DEM, CLCD, and building-footprint sample datasets used for reproduction are included for review purposes subject to the original data-use permissions. MODIS land surface temperature and AlphaEarth embedding products are accessed through Google Earth Engine and can be re-extracted using the documented scripts by users with Earth Engine access.
