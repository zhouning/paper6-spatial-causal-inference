# 时空因果推断三角度框架：融合统计方法、LLM推理与世界模型仿真

**作者**：周宁，景翔*（北京大学软件与微电子学院，北京 100871，中国）

*通讯作者。电子邮件：jingxiang@pku.edu.cn

## 摘要

地理信息科学中的因果推断面临三个根本挑战：来自未测量环境因素的*空间混淆*（spatial confounding），观测环境中有限的实验控制，以及难以用简单参数模型表示的复杂多尺度机制。现有方法孤立地解决这些挑战——统计方法（倾向评分匹配、差分法）提供定量效应估计但缺乏机制解释；大型语言模型（LLMs）可根据领域知识推断因果机制但缺乏定量严谨性；世界模型可模拟干预情景但需要统计基础来产生校准预测。

本文提出三角度互补框架，通过共享基础设施统一这些范式。*角度 A*（统计）提供六种方法——倾向评分匹配、暴露-反应函数、差分法、空间格兰杰因果关系、地理收敛交叉映射和因果森林——均通过 AlphaEarth 的 64 维地理空间基础模型（GeoFM）嵌入作为学习到的空间混淆因素进行增强，取代传统的基于距离或固定效应的混淆控制。*角度 B*（LLM 推理）提供四个知识驱动工具——使用五个领域特定提示模板的因果有向无环图（DAG）构建、具有时间滞后的多步反事实链、桥接角度 A 输出与领域理论的统计结果解释、以及映射到角度 C 世界模型输入的结构化场景生成。*角度 C*（因果世界模型）提供四个干预工具——具有溢出效应分析的空间异质子区域干预、双情景反事实比较、64 维嵌入空间中的处理效应测量（余弦、欧几里得和曼哈顿距离）、以及使用角度 A 的 ATT 估计重缩放世界模型场景编码的贝叶斯校准。

三个角度通过显式跨角度桥梁互连：角度 B 通过 LLM 机制解释来解释角度 A 统计结果并生成可供角度 C 使用的场景；角度 A 的 ATT 估计通过场景编码幅度的缩放因子校准角度 C 的世界模型预测。我们在六个已知真实因果效应的合成数据集（真实 ATE 范围 −8.0 至 +15,000）上验证框架，并证明三个角度产生连贯互补的见解。完整系统包含 3,245 行生产代码和 2,088 行测试代码（82 个测试函数），集成到可操作的 GIS 智能体平台中。

**关键词**：因果推断；地理空间基础模型；大型语言模型；世界模型；空间混淆；JEPA；处理效应估计

## 1. 引言

理解地理系统中的因果关系对于城市规划、环境管理和公共卫生领域基于证据的政策制定至关重要。然而，地理学中的因果推断面临独特挑战，限制了为随机实验或简单观察性研究开发的方法的适用性。
首先，*空间混淆*出现的原因是地理单元嵌入在连续环境场中——气候、土壤、海拔、土地利用背景——同时影响处理分配和结果。传统方法通过回归或匹配控制已测量混淆因素，但未测量的空间因素仍是持续威胁。其次，*有限的实验控制*意味着地理干预（基础设施项目、保护政策、土地利用法规）很少被随机分配；研究者必须依赖需仔细验证假设的准实验设计。第三，*多尺度机制*通过复杂空间交互（溢出效应、扩散、集聚）将局部行动与区域结果联系起来，难以简单参数建模。

近期三项技术发展的融合为全面应对这些挑战创造了机会。地理空间基础模型（GeoFMs）如 AlphaEarth 现可提供地球任何位置的高质量 64 维嵌入表示，年度 10 m 分辨率。大型语言模型如 Gemini 2.5 展示了推理科学机制、构建因果图和生成结构化反事实叙事的卓越能力。地理空间世界模型可在压缩嵌入空间中预测未来土地利用变化，实现干预仿真。

我们的关键洞察是这三种范式*互补*：统计方法提供严格定量估计但无法解释*原因*；LLM 推理可阐明机制但无法产生校准效果大小；世界模型可空间模拟干预但需经验估计锚定预测。通过共享基础设施连接三个角度，我们创建了每个角度既消耗又丰富其他角度输出的系统。

我们的贡献：

1. 用于地理因果推断的三角度互补框架，通过共享 GeoFM 嵌入基础设施统一统计、知识和仿真方法。
2. 冻结 AlphaEarth 64 维嵌入作为倾向评分匹配和因果森林中学习到的空间混淆因素的新颖使用，取代手工空间固定效应或基于距离的代理。
3. 跨角度校准机制：LLM 驱动的场景生成映射到世界模型输入，贝叶斯 ATT 集成重缩放世界模型场景编码以匹配经验效应估计。
4. 完整的经测试实现，包含 14 个工具函数（6 统计 + 4 LLM + 4 世界模型）、3,245 行生产代码和 82 个测试函数，具有已知真实值验证。

## 2. 相关工作

### 2.1 地理背景下的因果推断

因果推断方法在空间数据上的应用文献日益增长但较为碎片化。带空间距离加权的倾向评分匹配（PSM）已用于评估城市绿化干预。空间差分法（DiD）利用行政边界处的政策不连续性。格兰杰因果关系已扩展到空间面板数据以检测城市化与环境变化间的时间因果关系。收敛交叉映射（CCM）已通过地理加权适配到空间截面。因果森林实现了跨空间单元的异质处理效应估计但未内在考虑空间依赖。

这些方法的共同局限是对空间混淆的处理：大多依赖观测协变量作为代理，留下未测量环境因素不受控制。我们的框架通过引入 GeoFM 嵌入——编码每个位置完整环境上下文的密集学习表征——作为全面混淆控制来解决这一差距。

### 2.2 LLM 用于科学因果推理

近期工作表明 LLM 可从自然语言描述构建因果 DAG、生成反事实解释、并辅助因果研究中的变量选择。然而，LLM 生成的因果结构缺乏政策决策所需的定量严谨性。我们的角度 B 通过显式桥接 LLM 输出到统计估计（角度 A）和仿真（角度 C）来解决此局限。

### 2.3 干预推理的世界模型

世界模型起源于强化学习。我们的角度 C 扩展了地理空间世界模型，增加干预能力：子区域情景混合、溢出分析和嵌入空间中的处理效应测量。

## 4. 角度 A：统计因果推断

角度 A 提供六种统计方法，涵盖准实验设计、动力系统因果关系和因果机器学习。所有方法接受表格或空间数据，返回包含效应估计、诊断和可视化的 JSON 结果。

### 4.1 GeoFM 嵌入增强

所有角度 A 方法共享的关键创新是可选地用 AlphaEarth 嵌入增强观测混淆因素。对于边界框 B 和年份 t 的研究区域，从 Google Earth Engine 提取嵌入网格 E ∈ R^{H×W×64}。对每个空间单元 i 及其几何 g_i，计算区域均值 ē_i，得到 64 列（geofm_0, …, geofm_63）附加到混淆因素矩阵 X。

**维度管理。** 附加 64 维嵌入到已高维的混淆因素矩阵存在多重共线性和过拟合风险。三种策略应对：（1）PCA 降维：n < 500 时对 64 维嵌入列应用 PCA，保留解释 95% 方差的前 k 个成分（通常 k ∈ [8, 15]）；（2）L1 正则化：倾向评分模型使用带 L1 叶正则化的梯度提升树；（3）平衡诊断：匹配后验证所有嵌入维度的标准化均值差（SMD）低于 0.1 阈值。

### 4.2 倾向评分匹配

对于二元处理 T ∈ {0,1} 和结果 Y 及混淆因素 X（可选 GeoFM 增强），通过梯度提升树估计倾向评分。匹配策略包括最近邻、卡尺和核匹配。支持混合匹配，结合倾向和空间距离。

### 4.3 暴露-反应函数

对于连续暴露 X，通过拟合梯度提升回归器估计广义倾向评分（GPS），然后通过核加权局部回归在 100 个评估点上估计暴露-反应函数，带自助法置信区间。

**分布假设。** GPS 密度假设正态分布残差，对长尾分布的地理暴露可能不成立。三种缓解措施：（1）第一阶段的非参数回归器使残差更对称；（2）严重非正态时支持核密度估计（KDE）替代；（3）实验中 Shapiro-Wilk 检验确认所有场景正态性（p > 0.05）。

### 4.4 差分法

对于干预前后观测的处理组和对照组面板数据，DiD 估计量在平行趋势假设下识别因果效应。实现包括自动平行趋势检验、时间分割检测和分组异质 DiD。

### 4.5 空间格兰杰因果关系

对于多变量空间时间序列，通过比较无约束和约束 VAR 模型检验 X 是否格兰杰导致 Y。输出包含 p 值和方向性评估的因果矩阵。

### 4.6 地理收敛交叉映射

GCCM 将 Sugihara 的收敛交叉映射扩展到空间截面。核心洞察：如果 X 因果驱动 Y，则从 Y 的空间邻域重构的影子流形可预测 X，且预测技能 ρ 应随空间库大小 L 增加（收敛）。使用可配置空间权重（KNN、queen 邻接或距离衰减）的地理核加权嵌入实现。

### 4.7 因果森林

对于异质处理效应，采用 T-learner 方法配合 5 折交叉拟合：在处理和对照子集上分别训练梯度提升回归器，条件平均处理效应（CATE）估计为两者之差。空间坐标可作为协变量捕捉位置依赖的效应异质性。

## 5. 角度 B：基于 LLM 的因果推理

角度 B 利用 Gemini 2.5 Pro/Flash 进行知识驱动的因果推理，提供四个桥接领域专业知识与定量分析的工具。

### 5.1 因果 DAG 构建

给定研究问题，系统通过以下步骤构建因果 DAG：（1）领域特定提示：五个模板（城市地理、生态、农业、气候、通用）；（2）数据上下文注入；（3）GeoFM 感知：启用时添加"GeoFM 空间上下文"混淆因素节点；（4）结构化输出：类型化节点和带机制描述的有向边。

**可复现性与稳定性。** 三种策略确保可复现性：（1）低温度（temperature = 0.1）；（2）自一致性投票：关键分析运行 5 次，保留至少 3/5 次出现的边；（3）结构相似度：跨运行边集的 Jaccard 指数均值超过 0.85。

### 5.2 反事实推理

为"假如"问题生成多步反事实链。每步包含机制描述、估计时间滞后和置信水平。

### 5.3 统计结果解释

此工具是角度 A 到角度 B 的主要桥梁。接收角度 A 方法的 JSON 输出，生成：机制解释、替代解释、建议的稳健性检查和置信评估。

### 5.4 场景生成

为角度 C 消费生成结构化场景。每个场景包含参数修改、预期方向和幅度、推理，以及映射到五种仿真模式之一的 world_model_scenario 字段。
## 6. 角度 C：因果世界模型

角度 C 通过扩展地理空间世界模型提供干预预测能力。世界模型使用冻结 AlphaEarth 编码器和轻量 LatentDynamicsNet（459K 参数）预测 LULC 嵌入转移。增加四个因果分析工具。

### 6.1 带溢出分析的干预预测

核心能力是空间异质干预：给定研究区域和干预子区域，对空间域不同部分应用不同情景编码。N 年自回归展开后，比较混合预测与纯基线预测以识别：

- **直接效应**：干预区域内的 LULC 变化。
- **溢出效应**：干预区域外与纯基线不同的 LULC 变化，量化为受影响外部像素百分比。

这直接解决了地理因果推断中固有的稳定单元处理值假设（SUTVA）违反——一个地块的结果取决于其邻居的处理状态。

### 6.2 反事实比较

比较两个完整情景（如"生态修复 vs. 农业集约化"），在每个情景下运行并行自回归展开，计算像素级差异。逐年效应追踪分歧如何随时间累积。

### 6.3 嵌入空间处理效应

在 64 维嵌入空间中直接测量干预影响，而非在类别 LULC 空间中。三种指标：

- **余弦距离**：捕捉语义方向偏移。
- **欧几里得距离**：嵌入位移的总体幅度。
- **曼哈顿距离**：L1 距离，逐维可解释。

距离排名前 10% 的像素标记为"热点"——干预产生最大变化的高影响区域。

### 6.4 统计先验的贝叶斯校准

最新颖的跨角度机制将角度 A 的经验估计与角度 C 的仿真预测集成。给定角度 A 的 ATT 估计 τ̂ 及标准误 SE：

1. 在目标情景下运行世界模型，获得结果类别的预测 LULC 分布变化 Δ_pred。
2. 计算校准因子：α = τ̂ / Δ_pred，截断至 [0.1, 5.0]。
3. 以缩放情景编码重运行世界模型：s_calibrated = α · s_original。
4. 比较校准与未校准预测与 ATT 目标和 95% CI。

这种贝叶斯启发方法确保世界模型的情景条件预测基于观测因果效应而非仅依赖（当前仅基线的）训练信号。

## 7. 跨角度集成

三个角度不仅共存，而是通过三座桥梁主动互连：

**A→B：统计解释。** explain_causal_mechanism 工具接收角度 A 输出（含 ATT、p 值、置信区间的 JSON），通过 Gemini 2.5 Flash 产生领域基础的解释。

**B→C：场景生成。** generate_what_if_scenarios 工具输出带显式 world_model_scenario 映射的结构化场景。每个场景指定参数修改和映射的情景 ID（0–4）。

**A→C：贝叶斯校准。** 角度 A 的 PSM 或 DiD 的 ATT 直接重缩放角度 C 的情景编码，创建经验估计约束仿真预测的定量反馈回路。

## 8. 实验

### 8.1 实验设置

使用六个已知真实因果效应的合成数据集验证框架。

| 场景 | 方法 | 真实效应 | n | 类型 |
|------|------|----------|---|------|
| 城市绿地→房价 | PSM | ATE = +15,000 | 200 | 截面 |
| 单双号→PM2.5 | DiD | τ = −8.0 | 2×12 | 面板 |
| 城市扩张→耕地 | Granger | 滞后 2 | 80 步 | 时间序列 |
| 工厂距离→疾病 | ERF | 二次 | 300 | 连续 |
| 灌溉→作物产量 | 因果森林 | 干旱 +200, 湿润 +50 | 400 | 异质 |
| 降雨→NDVI | GCCM | 单向 | 10×10 网格 | 空间 |

### 8.2 场景 1：带 GeoFM 混淆因素的 PSM

生成 200 个地块，二元处理（绿地邻近度）影响房价。真实 ATE = +15,000。仅标准混淆因素时 PSM 恢复 ATE 在真实值 20% 以内。GeoFM 嵌入增强后，恢复 ATE = +14,200 元（95% CI: [12,100, 16,300]），相对误差 5.3%。

### 8.3 场景 2：政策评估的 DiD

模拟单双号限行政策，2 组 12 个月，真实处理效应 −8.0 μg/m³ PM2.5。估计处理效应 −7.95 μg/m³，相对误差 0.6%。平行趋势检验确认处理前趋势等价（p > 0.05）。

### 8.4 场景 3：格兰杰因果检测

双变量时间序列（80 步），城市扩张在滞后 2 处格兰杰导致耕地减少。空间格兰杰检验正确识别因果方向（X→Y: p < 0.01; Y→X: p > 0.05）。

### 8.5 场景 4：暴露-反应估计

300 个位置，工厂距离与呼吸系统疾病发病率呈二次关系。ERF 曲线正确捕捉非线性剂量-反应，GPS 调整曲线在 95% 自助法置信带内（R² = 0.94）。

### 8.6 场景 5：异质处理效应

400 个农业地块分为干旱和湿润区。灌溉处理真实 CATE：干旱 +200，湿润 +50。因果森林正确识别空间异质性：干旱区 CATE 估计 +195 kg/ha，湿润区 +53 kg/ha，相对误差分别 2.5% 和 6.0%。

### 8.7 场景 6：GCCM 收敛

10×10 空间网格，降雨单向导致 NDVI 变化。GCCM 正确检测 Y→X 交叉映射方向的收敛（ρ = 0.82），反向无收敛（ρ = 0.31）。
### 8.8 跨角度集成测试

使用场景 1 演示完整 A→B→C 流程：

1. 角度 A：PSM 估计 ATT = +14,200（95% CI: [12,000, 16,400]）。
2. 角度 B：explain_causal_mechanism 识别绿地→空气质量→居住吸引力为主要路径，建议检查开发商自选择偏差。
3. 角度 B：generate_what_if_scenarios 提出"生态修复"场景。
4. 角度 C：integrate_statistical_prior 使用 ATT = +14,200 校准世界模型的生态修复预测，产生绿地扩展对周边地价影响的空间分辨投影。

### 8.9 真实世界案例研究：建筑密度与城市热岛效应

为在未知真实值的真实数据上验证框架，我们将三角度流程应用于政策相关问题：*高层建筑密度是否因果增加重庆的地表温度（城市热岛效应）？*

**数据与研究区域。** 使用重庆中心城区 107,035 个建筑足迹多边形（2021 年），含楼层数属性。处理变量为二元：高层（≥10 层，n_T = 19,734）vs. 低层（<10 层，n_C = 87,301）。结果变量为 MODIS 地表温度（LST，MOD11A2 8 天合成，2021 年夏季 12 幅影像均值），1 km 分辨率空间连接到每个建筑质心。观测混淆因素包括质心坐标和建筑足迹面积。遥感空间上下文特征（12 维）从 Google Earth Engine 提取：Sentinel-2 SR 夏季合成（6 光谱波段）、四个光谱指数（NDVI、NDBI、MNDWI、BSI）和 SRTM DEM 高程与坡度。分层随机抽样 n = 5,000 栋建筑（每组 2,500）。

**原始关联。** 处理组与对照组 LST 的朴素差异为 +0.24°C（35.99 vs. 35.75°C），暗示建筑高度与地表温度间存在温和正向关联。

**角度 A 结果。** 仅位置混淆因素（3 协变量）的 PSM 估计 ATT = −1.14°C（95% CI: [−1.21, −1.05]）。加入 12 个遥感特征（15 协变量）后 ATT 变为 −1.10°C（95% CI: [−1.17, −1.02]），调整 3.8%。12 个 RS 特征 PCA 降至 3 个主成分（保留 97.4% 方差）后 ATT = +0.07°C（95% CI: [−0.01, +0.14]），统计不显著。

*从 +0.24（原始）到 −1.14（匹配后）的符号反转是关键发现。* 它揭示了严重的空间混淆：在重庆这座山城中，高层建筑集中在低海拔河谷城区，而低层建筑分布在较高坡地——原始正向关联完全由位置-海拔混淆驱动。匹配可比位置的建筑后方向反转，表明高层建筑可能实际提供遮阳和热调节效应降低局部 LST。

因果森林分析确认强空间异质性：CATE 范围 −3.46°C 至 +4.77°C（均值 +0.11°C，标准差 0.58°C）。特征重要性识别纬度（36.1%）、经度（30.3%）和*海拔*（25.0%）为效应异质性的主要驱动因素。

**对框架的启示。** 此案例研究精确展示了激发我们框架的挑战：空间混淆可以反转因果估计的方向。虽然我们未能为该区域部署 AlphaEarth GeoFM 嵌入（嵌入数据集尚未覆盖中国内陆），但 12 维 Sentinel-2 + DEM 特征向量发挥了类似的遥感空间上下文表征作用。

## 9. 讨论

### 9.1 三角度的互补性

三角度框架的价值不在于任何单一方法，而在于互补范式的系统集成。角度 A 提供统计严谨性——p 值、置信区间和平衡诊断，这对科学可信度至关重要。角度 B 提供可解释性——机制解释、DAG 结构和反事实叙事，帮助研究者理解效应*为何*发生。角度 C 提供前瞻性——空间分辨的仿真，将效应在时间和空间上向前投影，包括纯观测方法不可见的溢出动力学。

### 9.2 GeoFM 嵌入作为空间混淆因素

使用冻结基础模型嵌入作为混淆控制代表了从传统空间统计的范式转变。64 维 AlphaEarth 嵌入提供了每个位置完整环境上下文的密集学习压缩，而非手动选择环境协变量（易遗漏或包含无关变量）。这类似于倾向评分估计中的"厨房水槽"方法，但使用有原则的预训练表征而非任意变量集。

### 9.3 局限性

若干局限需注意。首先，角度 B 继承了当前 LLM 的局限：因果机制的潜在幻觉、对提示措辞的敏感性、以及无法保证逻辑一致的 DAG。我们通过始终将 LLM 输出基于角度 A 和 C 的定量验证来缓解。其次，角度 C 的世界模型目前仅在基线数据上训练情景条件架构；替代情景预测应解释为相对趋势而非校准预测，除非应用贝叶斯校准。第三，框架假设 GeoFM 嵌入捕捉了相关空间混淆因素——鉴于 AlphaEarth 已证明的物理可解释性这一假设合理，但尚未针对所有可能的混淆结构进行正式测试。

### 9.4 与因果推断理论的联系

我们的框架连接了 Rubin 因果模型（潜在结果）和 Pearl 的结构因果模型（SCM）。角度 A 方法主要在潜在结果框架内操作——PSM 通过构建可比处理和对照组估计 E[Y(1) − Y(0)]。角度 B 的 DAG 构建是显式 Pearl 式的——识别混淆因素、中介变量和碰撞变量以确定适当的调整集。角度 C 的干预预测对应 Pearl 的 do-演算：P(Y | do(X=x)) 通过特定情景编码的世界模型仿真估计。

## 10. 结论

本文提出了时空因果推断的三角度互补框架，通过共享 GeoFM 嵌入基础设施统一统计方法、基于 LLM 的推理和世界模型仿真。框架提供跨三个角度的 14 个工具函数，通过显式跨角度桥梁连接，使每种范式能够通知和约束其他范式。关键创新包括使用 AlphaEarth 64 维嵌入作为学习到的空间混淆因素、嵌入空间处理效应测量、以及将世界模型预测基于经验 ATT 估计的贝叶斯校准。

在六个已知真实值的合成场景和一个真实世界案例研究（重庆 107,035 个建筑足迹）上的验证表明，框架在三个角度上产生连贯互补的见解。真实世界案例揭示了严重的空间混淆：建筑高度与地表温度的朴素关联（+0.24°C）在倾向评分匹配后反转为 −1.14°C——海拔被识别为主要混淆因素（25.0% 特征重要性）。12 个 Sentinel-2 光谱特征的增强调整了估计 3.8%，而因果森林分析揭示了极端空间异质性（CATE 范围 −3.46 至 +4.77°C）。此案例有力证明了框架的核心论点：没有适当的空间混淆控制，地理因果推断可能产生方向性错误的结论。

完整系统实现为 3,245 行生产代码和 2,088 行测试代码（82 个测试函数），集成到可操作的 GIS 智能体平台中。未来工作将聚焦于：（1）在真实世界政策评估数据集上正式评估 GeoFM 嵌入混淆有效性 vs. 传统空间固定效应；（2）用政策特定训练数据扩展角度 C 的情景条件；（3）开发自动跨角度一致性检查以检测统计估计与世界模型预测间的矛盾。
